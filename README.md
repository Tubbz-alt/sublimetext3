## usage

```bash
git clone https://github.com/slackwarecn-slackbuilds/sublimetext3.git
cd sublimetext3
source sublimetext.info
wget $DOWNLOAD          # For x86
wget $DOWNLOAD_x86_64   # For x86_64
./sublimetext3.SlackBuild
```
